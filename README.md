# rem

Local-first human-agent memory system.

## Workspace

- Runtime/package manager: Bun
- Language: TypeScript
- Lint/format: Biome
- Testing: bun test

## Docs

- Product requirements: `docs/prd.md`
- Architecture/design: `docs/design.md`
- Canonical tech stack: `docs/tech-stack.md`

## Quick start

```bash
bun install
bun run typecheck
bun run test
```
